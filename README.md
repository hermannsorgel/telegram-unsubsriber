# Unsubscriber for Telegram channels

The extension chromium-based browsers that helps to clean up Telegram subscriptions and leave unwanted channels

# How to use
1. Open Telegram Web K.
2. Scan to see all your channel subscriptions.
3. Uncheck any channels you want to keep.
4. The extension leaves the rest.

Everything runs locally in your browser. No data is collected or sent anywhere except directly to Telegram.
For bulk or automated processing, consider using the official Telegram API with a dedicated script.
